# 2026streak

This repo is my daily streak playground for 2026.

## streak_cli (simple Python mini-project)
`streak_cli` is a tiny CLI that stores quick notes/check-ins to a local JSON file (`data.json`).

### Run

```bash
python -m streak_cli
```

### Example

- Choose **1** to add a note
- Choose **2** to list notes

### Tests

```bash
python -m unittest
```

## Why this exists
Consistency beats motivation. One meaningful improvement per day.
